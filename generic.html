Smart Garage Door

Life has no remote. Get up and change it yourself.

Overview

Coming to home in your vehicle and searching for your remote controller to open garage door or main gate. Is it hard to find? Or has it slipped in your pocket or lost? Is your family usually gets sick and tired of it? License plate detection system is solution for all your problem. The vehicle just have to stop in front of the garage and detect your license plate, verify with saved records and if verified doors will open and then you will not need your 315 or 433 MHz garage remote. This type of problem is frequently faced by many people in all regions. Using Raspberry Pi, single board microcontroller (taking example of Arduino) and other small component, these technologies are cheap and easily assembled solution to this problem.

Description

License plate detection system is a digital processing image which is commonly used in maintain law enforcements, traffic rules and various security purposes. This device will connect to USB camera although most surveillance cameras are IP cameras and if you want more security you can connect it to home Wi-Fi and you will receive notifications whenever door will open and close by installing small customised app.

But As per knowledge Python is the best for Raspberry Pi programming. For IOT and home automation projects, mostly raspberry pi is used and for automation best language is any scripting language. I think even you can control it by connecting to Google or Alexa which mean you can operate it with your mobile in case of any difficulty like not able to detect license plate because of poor visibility.

How it will Work?

When vehicle came to door it captures image and KNN (K-nearest neighbors algorithm) is used to extract License Number from it. As this process work in three stages which are pre-processing, plate localisation and license plate recognition system. Pre-processing is to convert image to greyscale which consider each pixel of image which ranges between 0 to 255 and blur the image using Gaussian Filter and find all the contour images (mainly used for shape recognition and detection). In license plate recognition system KNN is used to find all possible character in image and crop it to extract character and numbers. Each character and numbers in plate should be segmented using KNN to recognise using training data. Extracted license number is run through database containing registered number and when found match then it runs command to open the door.

  • RF 315 or 433 MHz receiver, transmitter for opening and closing of door
